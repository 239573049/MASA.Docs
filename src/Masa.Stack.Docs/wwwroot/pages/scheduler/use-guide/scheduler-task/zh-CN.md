任务为Job运行时. 产生的一次运行记录
#### 任务状态
- 成功：当任务正常运行完成，则视为成功。
- 失败：当任务运行出现异常，导致不能正常完成，则视为失败。
- 超时：当任务运行超过配置的超时时间且超时策略为超时后忽略时，视为超时。
- 超时后成功：当任务超时后，返回成功， 则视为超时后成功。
- 等待自动重试：当任务失败后，执行了失败策略，进行重试，在重试间隔中，会显示此状态。
- 等待运行：当阻塞策略是，等待上次运行完成时，当上次运行未结束出现新的任务，则会显示为等待运行状态。
